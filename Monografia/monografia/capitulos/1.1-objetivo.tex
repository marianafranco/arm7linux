\section{Objetivo} \label{sec:intro_objetivo}
\index{introducao!objetivo} 

O objetivo deste projeto de formatura é desenvolver um \emph{microkernel} para a placa experimental ARM Evalutator-7T, constituída de um processador ARM7 e de alguns periféricos simples.

O \emph{microkernel} implementa os mecanismos básicos de um sistema operacional, como o chaveamento de \emph{threads}, as chamadas de sistema e utiliza algumas rotinas para a comunicação com os periféricos da placa.

Além disso, foram criados alguns programas para testar e exemplificar o funcionamento do \emph{microkernel}. Entre esses programas, um simples terminal foi desenvolvido para a interação dos usuários com o sistema.
