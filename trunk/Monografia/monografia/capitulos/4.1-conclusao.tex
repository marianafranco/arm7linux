\section{Conclusão} \label{sec:conclusao}
\index{consideracoes!conclusao}

Através deste projeto de formatura foi possível desenvolver, como previsto, um \emph{microkernel} simples para a placa ARM Evaluator-7T.

O KinOS é um \emph{microkernel} que implementa, de maneira didática, os mecanismos básicos de um sistema operacional, como: chaveamento de threads, chamadas de sistema e rotinas de comunicação com os periféricos da placa (\emph{leds}, \emph{display}, botão, \emph{switches}). Além disso, o KinOS possui um simples terminal para a interação do usuário com o sistema e de onde é possível executar os programas adicionados ao \emph{microkernel}.
